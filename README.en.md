# notestudio

#### 介绍
NoteStudio是一款MarkDown文本编辑器，功能丰富，致力于提高文字创作者的工作体验。

#### 软件架构
采用流行的JavaFx框架搭建，具有美观，可移植性高的特点。


#### 安装教程

你可以在这里下载NoteStudio https://jhone666.github.io/notestudioserver/NoteStudio.dll ，下载完成后修改文件格式为zip，解压即可使用
